# 🔍 Website Evaluator - Chrome 확장 프로그램

웹사이트를 분석하고 평가하는 Chrome 확장 프로그램입니다.

## ✨ 기능

### 평가 기준
- **시각적 매력도** - 색상, 레이아웃, 이미지, 디자인 구조
- **사용자 체류도** - CTA 버튼, 콘텐츠 구조, 내부 링크, 헤딩
- **모바일 반응성** - Viewport 설정, 반응형 이미지, 미디어 쿼리
- **로딩 속도** - 스크립트 수, WebP 이미지, 지연 로딩
- **신뢰도** - HTTPS, About/Contact 정보, 소셜 미디어 링크

## 🚀 설치 방법

### Step 1: 파일 구조 확인
```
Chrome-extension-web/
├── manifest.json
├── popup.html
├── popup.js
├── background.js
├── content-script.js
└── README.md
```

### Step 2: Chrome에 설치

1. Chrome 주소창에 `chrome://extensions/` 입력
2. 우측 상단 **"개발자 모드" 활성화**
3. **"압축해제된 확장 프로그램 로드" 클릭**
4. 이 폴더(`Chrome-extension-web`) 선택
5. ✅ 설치 완료!

## 💡 사용 방법

1. Chrome 우측 상단에 확장 프로그램 아이콘 클릭 🧩
2. 평가하고 싶은 웹사이트 URL 입력
   - 예: `https://example.com`
3. **"평가" 버튼 클릭**
4. 평가 결과 확인 📊

## 📊 평가 결과

### 종합 점수
- 0/10 ~ 10/10 사이의 점수

### 상세 분석
- 각 항목별 점수 표시
- 진행 바로 시각화
- AI 기반 평가 의견

## 🔧 기술 스택

- **manifest.json** - Chrome 확장 프로그램 설정
- **popup.html** - 사용자 인터페이스
- **popup.js** - 웹사이트 분석 로직
- **background.js** - 백그라운드 작업
- **content-script.js** - 페이지 스크립트

## 📝 파일 설명

### manifest.json
- 확장 프로그램의 메타데이터
- 권한 설정
- 백그라운드 스크립트 등록

### popup.html
- 사이드바 UI
- URL 입력 필드
- 평가 결과 표시 영역

### popup.js
- URL 검증
- 웹사이트 분석 알고리즘
- 점수 계산
- 결과 표시

### 평가 알고리즘

#### 시각적 매력도
```
- 배경색 확인 (+1점)
- 이미지 수 분석 (최대 +2점)
- 섹션/레이아웃 구조 (최대 +2점)
```

#### 사용자 체류도
```
- CTA 버튼 수 (최대 +2점)
- 헤딩 구조 (최대 +2점)
- 텍스트 콘텐츠 길이 (최대 +2점)
- 내부 링크 수 (최대 +1점)
```

#### 모바일 반응성
```
- Viewport 메타 태그 (+3점)
- 반응형 이미지 (+1점)
- CSS 파일 존재 (+1점)
```

#### 로딩 속도
```
- 스크립트 수 (-최대 3점)
- WebP 이미지 (+2점)
- 이미지 지연 로딩 (+2점)
```

#### 신뢰도
```
- HTTPS 프로토콜 (+2점)
- About/Contact 정보 (+1.5점)
- 소셜 미디어 링크 (+1.5점)
```

## 🐛 제한사항

- CORS 정책으로 인해 일부 사이트는 분석 불가능
- 동적으로 로드되는 콘텐츠는 정확한 분석 어려움
- iframe을 통한 분석으로 일부 스타일이 제대로 감지되지 않을 수 있음

## 🔐 보안

- 로컬 분석만 수행 (서버로 데이터 전송 안 함)
- 사용자의 브라우징 데이터 저장 안 함
- 확장 프로그램 내부에서만 처리
